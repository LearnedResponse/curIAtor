# Skipped public backlog items

Retired public work-orders that are **not shipped**, but also should not remain in the active local
backlog. This is for work blocked on external publication/hosting steps, public repositories, DOI
services, third-party communities, or deliberately deferred post-release directions.

## Convention

1. Move the file here unchanged except for status/link fixes needed to make the retirement reason clear.
2. Move its index line in [`../README.md`](../README.md) to **## Skipped / External** with the reason.
3. Do not describe these as shipped. Reopen by moving the file back to the live backlog with a concrete
   local work-order.
